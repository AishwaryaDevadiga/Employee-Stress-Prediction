## 🚀 Deployment Instructions

### Local Development
```bash
# Step 1: Verify setup
python setup_verify.py

# Step 2: Train models (first time)
python train.py

# Step 3: Launch application
streamlit run hr_platform.py
```

### Quick Launch
```bash
# Use interactive menu
python launch.py
```

### Cloud Deployment

#### Option 1: Streamlit Cloud (Free)
1. Push to GitHub
2. Go to streamlit.io/cloud
3. Select repository
4. Click "Deploy"

#### Option 2: AWS
1. Create EC2 instance
2. Install Python 3.9+
3. Clone repository
4. Run: `streamlit run hr_platform.py --server.port 80`

#### Option 3: Docker
```dockerfile
FROM python:3.9
WORKDIR /app
COPY . .
RUN pip install -r requirements_streamlit.txt
CMD streamlit run hr_platform.py
```
